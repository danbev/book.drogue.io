Examples

There are several examples for different chips to show how drogue device can be used. The examples are grouped by how advanced they are, and what they demonstrate such as networking, but may overlap for examples that cover more than one area.

All example can be run using cargo run, which in some cases require the probe-run tool (cargo install probe-run) to flash the program.

Basic examples

Basic examples that only require a single board without any additional hardware.

  • Raspberry Pi Pico Blinky

  • Example application demonstrating how to write packages containing multiple actors

  • Ping pong show how to use circular dependencies between actors

  • STM32 Nucleo-H743 blink dance

  • STM32 Nucleo-H743 blinky

WiFi connectivity examples

Examples that demonstrates network connectivity using WiFi, usually using an additional network adapter module. May require setting up additional network services (locally or third party).

  • BBC micro:bit v2 + ESP8266 WiFi adapter

  • Example using a USB-to-Serial adapter + ESP8266 adapter

  • STM32 Nucleo-H743 + ESP8266 WiFi adapter

  • STM32 IoT Discovery board (iot01a) reporting sensor data using WiFi

LoRaWAN connectivity examples

Examples that demonstrates network connectivity using LoRaWAN. May require setting up additional network services (locally or third party).

  • BBC micro:bit v2 + RAK811 LoRa Adapter with AT command firmware

  • STM32 LoRaWAN Discovery board connecting to The Things Network

  • RAK811 Tracker Node connecting to The Things Network

  • STM32 Nucleo-WL55 using LoRaWAN connecting to The Things Network

BLE connectivity examples

Examples that demonstrates using Bluetooth Low Energy (BLE). May require running applications on a phone or PC.

  • BBC micro:bit v2 BLE temperature sensor

Drogue Cloud connectivity examples

Examples that demonstrate connectivity to Drogue Cloud.

  • BBC micro:bit v2 + ESP8266 WiFi adapter

  • Example sending telemetry data from Drogue Device to Drogue Cloud

  • Example using a USB-to-Serial adapter + ESP8266 adapter

  • STM32 Nucleo-H743 + ESP8266 WiFi adapter

  • STM32 IoT Discovery board (iot01a) reporting sensor data using WiFi

Display examples

Examples that demonstrates the use of displays, either on board or as external peripherals.

  • BBC micro:bit v2 uart and matrix display

  • STM32 Nucleo-H743 7-color e-paper display

UART examples

Examples that demonstrates the use of a UART.

  • BBC micro:bit v2 uart and matrix display

Other examples

Other examples that does not belong in any major category
